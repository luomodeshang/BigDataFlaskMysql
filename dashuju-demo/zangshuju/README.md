# 工业大数据分析 - 脏数据演示平台

这是一个基于Flask的可视化演示平台，用于展示工业大数据分析中的四种脏数据处理方法。

## 功能特性

本平台演示了以下四种脏数据处理方法：

1. **数据缺失 (Missing Data)**
   - 展示正常数据和包含缺失值的数据对比
   - 显示缺失数据点数量和缺失率
   - 使用虚线标识缺失数据

2. **数据滤波 (Data Filtering)**
   - 展示原始噪声数据、干净信号和滤波后的数据
   - 使用移动平均滤波算法去除噪声
   - 清晰展示滤波效果

3. **离群值 (Outliers)**
   - 使用3σ原则（均值±3倍标准差）检测离群值
   - 展示正常数据、包含离群值的数据和检测到的离群值
   - 显示统计信息和检测阈值

4. **归一化 (Normalization)**
   - 展示不同量纲的原始数据（温度、压力、湿度、转速）
   - 提供Min-Max归一化和Z-score归一化两种方法
   - 可以切换查看不同归一化方法的效果

## 安装和运行

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行应用

```bash
python app.py
```

### 3. 访问应用

在浏览器中打开：`http://localhost:5000`

## 技术栈

- **后端**: Flask (Python Web框架)
- **数据处理**: NumPy, Pandas
- **前端可视化**: Chart.js
- **样式**: 原生CSS

## 使用说明

1. 启动应用后，在浏览器中访问主页
2. 点击顶部标签页切换不同的脏数据类型
3. 每个页面都提供"刷新数据"按钮，可以生成新的随机数据
4. 归一化页面可以通过下拉菜单切换不同的归一化方法

## 数据说明

- **数据缺失**: 模拟30天的温度数据，随机缺失20%的数据点
- **数据滤波**: 模拟7天的温度数据，包含噪声和滤波后的结果
- **离群值**: 模拟30天的温度数据，包含5%的离群值
- **归一化**: 模拟30天的多维度数据（温度、压力、湿度、转速）

## 注意事项

- 数据是随机生成的，每次刷新都会得到不同的结果
- 图表使用Chart.js库进行渲染，需要网络连接加载CDN资源
- 建议使用现代浏览器（Chrome、Firefox、Edge等）访问

## 教学用途

本平台专为工业大数据分析课程设计，可以帮助学生：
- 直观理解各种脏数据的特点
- 学习数据预处理的基本方法
- 通过可视化对比理解处理效果
