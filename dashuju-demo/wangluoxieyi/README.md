# 网络协议可视化演示平台

这是一个用于工业大数据分析教学的网络协议可视化演示平台，通过直观的可视化方式展示USB、TCP、UDP、MODBUS四种协议的特点和适用场景。

## 功能特点

- **USB协议演示**：展示串行总线、点对点连接、高速传输特点
- **TCP协议演示**：展示可靠传输、面向连接、流量控制特点
- **UDP协议演示**：展示快速传输、无连接、可能丢包特点
- **MODBUS协议演示**：展示工业协议、主从模式、请求-响应特点
- **实时数据更新**：动态展示协议运行状态
- **对比分析**：雷达图对比四种协议的特性

## 安装和运行

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行应用

```bash
python app.py
```

### 3. 访问应用

在浏览器中打开：http://localhost:5000

## 技术栈

- **后端**：Flask
- **前端**：HTML5, CSS3, JavaScript
- **图表库**：Chart.js
- **可视化**：CSS动画、Canvas图表

## 使用说明

1. 启动应用后，页面会显示四个协议卡片
2. 每个卡片包含：
   - 协议连接示意图
   - 实时数据指标
   - 动态图表
   - 协议特点标识
3. 数据每2-3秒自动更新一次
4. 页面底部有协议对比雷达图
